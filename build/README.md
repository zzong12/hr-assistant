# Build Resources

Place app icons here for electron-builder:

- `icon.icns` - macOS icon (1024x1024, .icns format)
- `icon.ico` - Windows icon (256x256, .ico format)
- `icon.png` - Fallback icon (512x512 PNG)

You can generate these from a single 1024x1024 PNG using tools like:
- https://www.electron.build/icons
- `npm install -g electron-icon-builder`
